# KNX Integration Lab

このリポジトリは、個人のスマートホーム研究および転職ポートフォリオとしての「KNX ミニラボ」を構築するためのドキュメントをまとめたものです。KNX と既存のスマートホーム機器（Hue、Oura など）の連携を通じて、システム設計の理解と実装を両立させることを目指します。

## 目的

*   KNX アーキテクチャとグループアドレス設計を学ぶ
*   KNX IP通信とAPI連携の実践を行う
*   スマートホーム統合の設計ドキュメントを作成する
*   実験を通じてポートフォリオを構築する

## リポジトリ構成

*   docs/：プロジェクトに関する詳細な資料を格納するディレクトリ
    *   docs/knx-mini-lab.md：KNXミニラボの構成と設計指針
    *   docs/tasks.md：今後のタスクや研究メモを整理します

## 背景

KNXは欧州発のビル／住宅向けオープン規格であり、配線方式（TP）とIP方式で機器同士が通信します。国内ではまだ一般的ではありませんが、高級住宅や商業施設などで採用されています。

本プロジェクトでは、KNXを利用したスマートホーム設計を学び、実際に小規模なラボを構築することで、施工以外の設計・統合領域に焦点を当てています。

## 方針

*   理論の理解とETSの習熟を優先する
*   机上で完結するミニラボを構築し、壁内配線には手を付けない
*   KNX IP経由で外部システムと連携し、HueやOuraなど既存のデバイスを統合する
*   学習内容や設計ノウハウをドキュメント化し、ポートフォリオとして整理する
